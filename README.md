# SL40
40W warm and cool color fill light

## Introduction

SL40 is a 40W warm and cool color fill light. It is designed to be used in a variety of lighting environments, including indoor and outdoor spaces, and can be controlled via a mobile app or a remote control. SL40 features a 12V DC motor and a 12V DC fan, which can be used to create a warm and cool effect. The light can be dimmed using a push button, and can be programmed to change colors and brightness over time.

## Features

- 40W Warm and Cool Color Fill Light
- 12V DC Motor and Fan
- Mobile App Control
- Remote Control
- Dimming Function
- Programmable Color and Brightness
- Built-in LED
- 12V DC Power Supply
- Dimensions: 100x100x100mm
- Weight: 1.5kg

## Pricing

SL40 is currently available for purchase at a cost of $199.

## Availability

The SL40 can achieve a power of 30w for single color temperature and a peak of 50w for dual color temperature, but it is not recommended to keep it on for extended periods. Currently, the fan starts at its highest speed upon startup, and further optimization will be implemented in the future. The currently released version is solely for testing purposes.
